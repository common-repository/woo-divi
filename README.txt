== Description ==
This is an extension for the Divi Builder. You would require Divi Builder and WooCommerce to be present. This will add a button to the Divi Builder similar to Call to Action button and let's you add an Add to Cart button anywhere in a page for Simple Product. The support for Variable product is not yet present and will be added later.

== Installation ==


1. Upload `woodivi.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. A new divi module will be added with name WooCommerce Product
